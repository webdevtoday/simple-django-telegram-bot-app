# Django + Telegram Bot

## Setup

Setup venv
```
python3 -m venv venv
```
Install dependencies
```
pip -r install pip-requirements.txt
```
Paste Telegram Bot TOKEN into tga/tga/settings.py
```
...
TOKEN = '<TOKEN>'
...
```
Run server
```
python tga/manage.py runserver
```

user
